laravel

historique

Laravel a été créé par Taylor Otwell en juin 20112. Le référentiel Laravel/laravel présent sur le site GitHub contient le code source des premières versions de Laravel. À partir de la cinquième version, le framework est développé au sein du référentiel Laravel/framework

installation

Avant de créer une nouvelle application Laravel sur votre ordinateur Windows, assurez-vous d’installer Bureau Docker. Ensuite, vous devez vous assurer que le sous-système Windows pour Linux 2 (WSL2) est installé et activé. WSL vous permet d’exécuter des exécutables binaires Linux en mode natif sur Windows 10. Vous trouverez des informations sur l’installation et l’activation de WSL2 dans microsoft documentation de l’environnement de développement.

Créer un nouveau projet

    composer create-project laravel/laravel example-app
Run un projet

cd example-app php artisan serve
